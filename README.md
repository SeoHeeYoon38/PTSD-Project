**게임컨셉 : 몽중몽**

> 끝없이 반복되는 악몽에서 탈출하고자 하는 퇴역 군인  
> 전쟁 트라우마를 자극하는 기묘한 연출  
> 총으로 모든 것을 해결함


📌 프로젝트 개요
- **개발 기간:** 2025.05.03 ~ 2025.05.18 (약 2주 진행)
- **개발 환경:** Unity, C#, Visual Studio
- **참여 인원:** 기획 1, 아트 2, 프로그래밍 4



🛠 주요 역할  
본 프로젝트에서 저는 **서브 프로그래머**로서 게임의 연출 로직과 시스템 데이터 연동을 담당했습니다.

- **GUI 및 자막 시스템:** CSV(엑셀) 데이터 파싱을 통해 게임 내 대사와 자막을 자동으로 출력하는 텍스트 처리 시스템 구현
- **이상현상 및 연출 로직:** 사이클 시작 시 사이렌/조명 트리거, 오브젝트의 비행 및 크기 변화, 화면 페이드 아웃 등 몰입감을 위한 연출 로직 개발
- **상호작용 트리거 시스템:** 총알 충돌을 통한 유리 파괴, 사격 광원을 활용한 이상현상 탐지 및 소멸(3회 타격), 버튼 클릭 시 징표 증식 기믹 등 핵심 게임 로직 구현
- **Git 활용 협업:** 팀 레포지터리에서 브랜치를 관리하며 지속적인 커밋과 코드 병합을 통해 프로젝트 형상 관리에 참여
